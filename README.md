# AdventureWorks Sales Analysis

## Overview
This project contains a sales analysis of the AdventureWorks dataset created in Microsoft PowerBI.

The analysis focuses on:
- sales performance across years and countries,
- customer segmentation,
- product-level performance insights.

The report covers data from 2020–2022 and uses interactive dashboards, slicers, DAX measures, and calculated metrics.

---

# Dashboards

## 1. General Dashboard
Main sales overview dashboard with:
- KPI switch between:
  - Profit
  - Quantity Sold
  - Sales/Revenue
- Year slicers (2020–2022)
- Total KPI comparisons
- YoY and MoM analysis
- YTD vs PYTD comparisons

---

## 2. Customer Dashboard
Customer-focused analysis including:
- Total customers
- Average revenue per customer
- Top 5 customers by revenue
- Customer segmentation by:
  - income level,
  - parent/non-parent status,
  - country
- New vs returning customer comparison

---

## 3. Product Dashboard
Product performance analysis including:
- Top 10 products by revenue
- Scatter analysis of top 25 high-revenue products:
  - X-axis → Profit Margin
  - Y-axis → Return Rate
- Quantity sold by category
- Average order value analysis
- KPI cards for:
  - most sold products,
  - most returned products,
  - highest profit margin subcategory

---

## Tools Used
- Microsoft PowerBI
- DAX
- Interactive dashboards and visualizations
